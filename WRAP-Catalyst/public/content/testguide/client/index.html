端末の設定

テスト端末の構築にあたっての端末設定方法です。こちらの設定はWindows/Mac共通です。
ブラウザごとに必要な設定は各ブラウザの項目をご参照ください。

ディレクトリ構成

必要なファイルと配置の構成は以下の通りです。
ディレクトリの配置場所は任意です。 

root/  
　├ node.json　Selenium WebDriverの設定ファイル
　├ selenium-server-standalone-x.x.x.jar　Selenium WebDriver本体
　└ 各ブラウザごとに必要なドライバ　Selenium WebDriverから各ブラウザを操作するためのドライバ

ツールのインストール

下記ツールを端末にインストールします。

ツール バージョン
Java 8～
Selenium WebDriver 3.4～

  • Java
    Java公式サイトのインストーラよりインストールしてください。

  • Selenium WebDriver
    Selenium公式サイトより、Selenium Standalone Serverをダウンロードし、任意のフォルダに配置してください。
    ※ Hubサーバで使用するjarファイルと同一のものです。

Selenium WebDriverの設定ファイル作成

node.jsonのファイル名で設定ファイルを作成し、Selenium WebDriverと同一のフォルダに配置してください。

  • node.jsonサンプル 
{
  "capabilities":
    [
      {
        "platform": "WIN10",
        "browserName": "chrome",
        "maxInstances": 2,
        "version": "58",
        "seleniumProtocol": "WebDriver"
      },
      {
        "platform": "WIN10",
        "browserName": "MicrosoftEdge",
        "maxInstances": 2,
        "version": "38",
        "seleniumProtocol": "WebDriver"
      },
      {
        "platform": "WIN10",
        "browserName": "internet explorer",
        "maxInstances": 2,
        "version": "11",
        "seleniumProtocol": "WebDriver"
      }
    ],
  "hub": "http://pt-wrapgen02-vmg.wni.co.jp:4444/grid/register"
}
  • 項目詳細
項目 内容
platform 端末名
browserName ブラウザ名
maxInstance 同時にテスト可能な数
version ブラウザバージョン
seleniumProtocol 「WebDriver」と記載
hub HubサーバのURL

platformとbrowserNameは正しく名称を記載しないとSeleniumが認識しませんので、ご注意ください。

  • platform
    各OSごとに以下の設定をplatformの項目に記載してください。
OS 設定
Windows 7 WIN7
Windows 8 WIN8
Windows 10 WIN10
Mac MAC
  • browserName
    ブラウザごとに以下の設定をbrowserNameの項目に記載してください。
ブラウザ 設定
IE internet explorer
Edge MicrosoftEdge
Chrome chrome
Safari safari

  • version
    数値で指定するとエラーになりますので、"（ダブルクォーテーション）で囲んで文字列として記載してください。
    必ずしもブラウザのバージョンと同一である必要はなく、必要に応じてブラウザのバージョン以外の情報を追記して構いません。

  • hub
    http://サーバのIPアドレス:4444/grid/registerを入力してください。
    ※HubサーバでSelenium Gridを起動時にコンソールに表示されます。

Selenium WebDriverの起動

各ブラウザで必要なドライバを配置し、Hubサーバ（Selenium Grid）を起動後、selenium-server-standaloneのjarファイルを起動します。
ドライバの配置方法は各ブラウザの章をご参照ください。

java -jar selenium-server-standalone-x.x.x.jar -role node -nodeConfig node.json

コンソールにThe node is registered to the hud and ready to useと表示されることを確認してください。
