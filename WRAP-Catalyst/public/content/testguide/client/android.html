WRAP-TestSuite（開発者向け）

Android

Android端末でテストを実行するための環境を構築します。

テスト端末

chromeが動作するAndroid端末を使用してください。 テストコードを実行する端末はWindows/Macのいずれでも使用できます。

Android端末設定

Android端末の設定画面から開発者向けオプションのUSBデバッグをオンにしてください。
※開発者向けオプションの出し方はAndroidのバージョン等によって異なるため各自検索してください。 基本的には[設定]→「端末情報」→「ビルド番号」を数回タップすると出てきます。

環境準備

  1. テストコードを実行するPCでAndroidSDKをダウンロードし、インストールをしてください。
  2. 下記のパスを環境変数に追加してください。
  3. ANDROID_HOME
    -Android SDK Toolsが置かれている場所
  4. PATH
    -Android SDK Toolsの場所を追加
    ・Mac,Linuxの場合
    $ANDROID_HOME/tools
    ・Windowsの場合
    %ANDROID_HOME%\tools

USB接続

  1. USBケーブルでPCとAndroid端末を結線してください。
  2. PCでUSBドライバを認識させてください。
  3. Android SDK Tools のadbコマンドで接続確認をしてください。
    % adb devices
List of devices attached
EP73216535 device

ドライバの配置

Chrome用のドライバであるChromeDriverを端末に配置します。
ダウンロードサイトより、ChromeDriverをダウンロードし、Selenium WebDriverと同一のフォルダに配置してください。
Chromeの対応バージョンが記載されていますので、テスト端末にインストールされているChromeのバージョンを確認の上、対象のドライバをダウンロードしてください。

chromeドライバの起動

下記コマンドを実行し、chromeドライバを起動してください。

chromedriver.exe --whitelisted-ips --url-base=wd/hub

テストコードのcapabilities

テストコードに記載されているcapabilitiesに下記の設定を追記してください。

const capabilities = {
    browserName: process.env.browserName,
    version: process.env.version,
    platform: process.env.platform,
    chromeOptions: {
        args: ['disable-infobars'],
        androidPackage: 'com.android.chrome'
    },
};
