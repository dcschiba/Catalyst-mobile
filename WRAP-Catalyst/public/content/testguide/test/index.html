テストコードの作成

テストコードについて

Seleniumのテストコードは以下の言語で記述することが可能です。

  • C#
  • JavaScript
  • Java
  • Python
  • Ruby

WRAP-TestSuiteではJavaScriptを標準として環境の構築やサンプルコードを作成していますので、テストコードの作成はJavaScriptで実施してください。
