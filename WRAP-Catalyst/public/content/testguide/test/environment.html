WRAP-TestSuite（開発者向け）

テストコードの作成環境

作成環境

端末はWindows/Macどちらでも可能です。下記ツールを端末にインストールしてください。

ツール 概要
node.js Seleniumのテストコードを実行します。
ImageMagick テスト結果のキャプチャ画像の加工や比較を行います。

テンプレートのダウンロード

WRAPの提供するテストコードのテンプレート一式をGitHubよりダウンロードし、任意のディレクトリに展開します。

ブラウザ設定の変更

テンプレート同梱のnode.jsonを開き、各ブラウザのversionを必要に応じて修正してください。

Selenium Gridのインストール

下記ツールを端末にインストールします。

ツール 概要
Java Selenium Gridを実行します。
Selenium Grid テスト端末へテストの命令を送信します。
  • Selenium Grid
    Selenium公式サイトよりSelenium Standalone Serverのjarファイルをダウンロードし、node.jsonと同一のフォルダに配置します。

Selenium Gridの起動

selenium-server-standaloneのjarファイルを起動します。

java -jar selenium-server-standalone-x.x.x.jar -role hub

クライアントサーバの起動

java -jar selenium-server-standalone-3.4.0.jar -role node -nodeConfig node.json

Selenium Grid→クライアントサーバの順に起動する必要がありますのでご注意ください。

ブラウザごとの設定

ブラウザごとに設定やdriverの配置が必要となるため、端末の設定の章をご参照ください。

テストコードの実行

テストコードが同梱されているpackage.jsonを開き、scriptsに設定されているversionを上記node.jsonに設定した値と揃えます。 

"scripts": {
  "test-mac-safari": "cross-env browserName=safari version=10 platform=MAC mocha index.js --timeout 60000",
  "test-mac-chrome": "cross-env browserName=chrome version=58 platform=MAC mocha index.js --timeout 60000",
  "test-win-chrome": "cross-env browserName=chrome version=58 platform=WIN10 mocha index.js --timeout 60000",
  "test-win-ie": "cross-env browserName=\"internet explorer\" version=11 platform=WIN10 mocha index.js --timeout 120000",
  "test-win-edge": "cross-env browserName=MicrosoftEdge version=38 platform=WIN10 mocha index.js --timeout 60000"
}

package.jsonが配置されているディレクトリ内で以下のコマンドを実行することでテストコードを実行できます。 

npm run test-win-chrome

自端末のブラウザが立ち上がり、テストコードが実行されます。
